# KORIX3D

## Uruchomienie lokalne

1. Skopiuj `.env.example` jako `.env.local`.
2. Uzupełnij adres projektu i publiczny klucz Supabase.
3. Uruchom `npm ci`, a następnie `npm run dev`.

Kompilacja produkcyjna: `npm run build`.
